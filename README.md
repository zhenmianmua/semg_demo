# sEMG 手势识别 — 信号预处理与可视化管线

## 一句话概述

基于 NinaPro DB5 基准数据集，构建 sEMG（表面肌电）信号的**预处理管线**与**多维度可视化系统**，为后续手势分类模型提供干净的输入数据。

---

## 背景与问题

sEMG 信号是非侵入式人机接口的核心载体，可应用于智能假肢、VR/AR 手势交互、康复训练等场景。但原始 sEMG 信号含有多种噪声和干扰：

| 噪声源 | 频率特征 | 典型来源 |
|---|---|---|
| 基线漂移 | < 5 Hz | 皮肤-电极界面 |
| 工频干扰 | 50 Hz | 交流电网 |
| 运动伪影 | < 20 Hz | 肢体晃动 |
| 高频噪声 | > 500 Hz | 电磁干扰 |

**本项目解决的核心问题**：如何在保留有用肌电信号的同时，系统性地去除上述干扰。

---

## 数据集：NinaPro DB5

- **来源**：NinaPro（Non-Invasive Adaptive Prosthetics）第 5 版，PLOS One 2017
- **采集设备**：2 个 Thalmic Myo 臂环（近端 + 远端），共 16 通道，200 Hz 采样
- **受试者**：10 名健全受试者，每人完成 52 种手势 × 6 次重复
- **当前演示数据**：Exercise 3（基础手指动作，12 种手势）

---

## 预处理管线

```
原始信号 (16ch, 200Hz)
    │
    ▼
┌─────────────────────────────┐
│ 4阶 Butterworth 带通 20–95Hz │  ← 去除基线漂移 + 高频噪声
└─────────────────────────────┘
    │
    ▼
┌─────────────────────────────┐
│ IIR 陷波器 @ 50Hz, Q=15      │  ← 去除工频干扰
└─────────────────────────────┘
    │
    ▼
┌─────────────────────────────┐
│ StandardScaler 逐通道归一化   │  ← 消除通道间幅值差异
└─────────────────────────────┘
    │
    ▼
干净信号 → 特征提取 / 模型输入
```

**关键发现**：原始参考代码将带通高截止设为 500 Hz，但采样率仅 200 Hz（奈奎斯特 = 100 Hz），导致程序崩溃。修复为 20–95 Hz 带通。
